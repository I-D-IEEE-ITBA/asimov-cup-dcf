# Display Contador Futbol (DCF)

## Descripción

Proyecto del *Display Contador Futbol*, que forma parte del desarrollo de la [Asimov Cup](https://ieeeitba.org.ar/asimovcup) en la competencia de *Futbol*.

DCF permite dar la señal de inicio del partido, cronometrar el tiempo de juego, pausarlo de ser necesario, a la vez que mantiene la cuenta de goles anotados por cada equipo.

El repositorio incluye tanto el hardware como el firmware desarrollado.
